## Installation

Gradle

```groovy
compile 'com.cipherthinkers:shapeflyer:1.0.0'
```

Maven
```xml
<dependency>
  <groupId>com.cipherthinkers</groupId>
  <artifactId>shapeflyer</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
